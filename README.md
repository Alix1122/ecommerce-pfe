# Projet E-commerce - PFE

## Description
Site e-commerce développé avec PHP, MySQL, Bootstrap.

## Fonctionnalités
- Catalogue produits
- Panier d'achat avec tailles (XS, S, M, L, XL, XXL)
- Connexion / Inscription
- Validation de commande
- Interface responsive

## Installation
1. Installer XAMPP
2. Copier le dossier dans `htdocs`
3. Importer `database/ecommerce.sql` dans phpMyAdmin
4. Accéder à `http://localhost/ecommerce-3.0/front/index.php`
